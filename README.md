 # Prerequisite
 Python is prerequirement and should be installed. (latest version or Python 3.10)
 
 # Steps to be followed
 1. Download the git folder
 2. Unzip it
 3. Run "pip install -r requirements.txt" in terminal
 4. Run "python main.py"
